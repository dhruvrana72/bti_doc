# Support

## Overview

The EXL Transaction Insights trial environment is designed for an intuitive and seamless user experience. However, users may occasionally encounter issues related to file formatting, processing delays, or account access. This section provides guidance on common issues and how to resolve them quickly.

---

## Common Issues and Solutions

### File Upload Errors

Possible Causes:
- Incorrect file format or missing required columns (e.g., transaction date or amount)
- File exceeds maximum allowed size for the trial version
- Encrypted or password-protected PDF file

Recommended Resolution:
- Re-upload the file in one of the supported formats (CSV, JSON).
- Ensure all required columns are present and headers are properly labeled.
- Refer to the Input Data Format Guide within the portal for correct schema and examples.

---

### Processing Delays

Possible Causes:
- High system traffic during peak testing hours
- Large or complex datasets with multiple accounts

Recommended Resolution:
- Allow a few minutes for completion and refresh the dashboard.
- If delay persists beyond 10 minutes, contact the support team via the Help section

---

### Output Not Displayed or Incomplete

Possible Causes:
- Partial data upload or interrupted processing
- Missing key fields such as transaction date or amount

Recommended Resolution:
- Re-upload the dataset ensuring all fields are complete.
- Verify that the uploaded file meets the minimum field requirement mentioned in the documentation.

---

### Contacting Support

If the issue persists or requires technical investigation, please reach out through any of the following channels available on the portal:
- Support Form: Submit an issue ticket directly from the Help section.
- Email Support: [mailto:support@exltransactioninsights.com] (support@exltransactioninsights.com)
- Response Time: All trial-related queries are typically addressed within 24 business hours.

The EXL support team ensures timely resolution and provides guidance to help you make the most of your trial experience.
